# 💧 Automated Water Level Controller PCB Design

## 📖 Project Overview

This project presents the design and development of an **Automated Water Level Controller** using **KiCad**. The main objective is to automatically monitor and control the water level in a storage tank, reducing water wastage and minimizing manual intervention.

As part of this project, I successfully designed the complete **PCB layout from scratch** using **KiCad**, including schematic capture, component placement, PCB routing, and design verification.

## 🎯 Objectives

* 💧 Automate water level monitoring and control.
* 🚰 Prevent water overflow and dry running of the motor.
* 🖥️ Design a compact and reliable PCB.
* 📚 Gain practical experience in PCB design using KiCad.

## 🛠️ Tools Used

* 🖥️ **KiCad** – Schematic capture and PCB layout design.
* ⚡ PCB Design Techniques – Component placement, routing, and design verification.

## ✨ Project Features

* ✅ Automatic water level monitoring.
* 🔷 Custom PCB designed from scratch.
* 📐 Organized component placement and PCB routing.
* ⚙️ Compact and efficient PCB layout.
